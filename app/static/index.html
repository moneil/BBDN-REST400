About this Demo

You are here because you are interested in the Blackboard REST APIs. This page is either being read from the source (good on you!), in a test environment on your local hardware, or on-line delivered through any number of server options (we will be using Heroku).

While this demo is written in Python the principles will apply to any language. And we have developed several demos in languages other than Python. See the below resources section for project requirements and useful links.

In the following sections we will show how to setup your local development environment, prepare a project, develop and run the project locally, and then push the project to a remote testing environment. All done using freely available resources.

Super Quick Start

Anxious to get a peek at the demo before you spend the time setting up your development environment? No problem!!

Create a Heroku account at heroku.com (it is free!). You can then try out the demo app by deploying it to your Heroku account simply by clicking the deploy button:




After deployment is complete you can customize the app from the Heroku git repository that is created.

Quick Start

Not quite ready to create a Heroku account? No problem!! Flask can run the application on your desktop!

If you have already pulled the source you can skip step 1. and proceed to step 2.

1. Open your terminal and navigate to the directory you want to contain your working directory. Execute these commands: 

$git clone git@github.com:blackboard/BBDN-REST400.git
$cd BBDN-REST400

2. While still in your terminal ensure you are running Python 2.7.11 (Python 3.x will not work) and then execute: 
$pip install -r requirements.txt
$python run.py
Then open your browser and navigate to http://localhost:5000/is_up
If all went well you will see "The BBDN-REST400 Demo is running." in your browser.

http://localhost:5000/ will load the index page which provides the launch points for the remaining setup and demos.

Requirements and Resources

Requirements

The document requirements.txt provides all the dependencies for deploying the application. In Brief:

  • Python 2.7.11 - Language version
  • Flask 0.10.1 - Controller
  • Bootstrap - UX framework
  • pyLTI 0.1.3 - LTI library
  • Jinja2 2.7.3 - Templating
  • Werkzeug 0.9.6 - Web Server Gateway Interface
  • gunicorn 19.1.1 - Web Server Gateway Interface (WSGI) HTTP server
  • oauth 1.0.1 - OAuth 1.0 libs for LTI
  • oauth2 1.5.211 - OAuth 2.0 libs for REST APIs
  • Flask-WTF 0.10.2 - Flask forms helper
