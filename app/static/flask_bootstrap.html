Flask and Bootstrap

We mentioned that two best practices for RESTful development is statless and responsive design. This demo uses two open projects and their libraries to achieve both - Flask and Bootstrap.

Flask and Bootstrap

Flask

Flask is a modular MVC framework - there are other popular frameworks such as Django. Flask was chosen for its modulararity and simplicity. Flask uses a template renderer called Jinga for delivery of pages populated with REST requested data.

More on Flask

Bootstrap

Bootstrap is a responsive design library originally developed by Twitter which provides the CSS libraries which facilitate building of responsive web pages. There are many design tools available to assist with responsive page layout - licensed and free. This demo was built using Pingendo.
